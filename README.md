# API_B3 - DOCKX Sven

### Importer le fichier postman_collection.json sur Postman afin d'avoir tout les endpoints pour tester l'API ainsi que des exemples pour chacun!

<br>

Sujet : https://docs.google.com/document/d/1Gw1xeXGR6guHCVzdM6iiY_CKWIR_BIpaWq-HTsBWuZg/edit

Informations:

Pour créer une entreprise, utiliser le endpoint /api/v1/auth/companyCreation

Pour créer un Freelance, utiliser le endpoint /api/v1/auth/freelanceRegister

Pour créer un Recruteur, utiliser le endpoint /api/v1/auth/recruiterRegister (l'entreprise renseigné pendant la création du recruteur doit exister)
